# AI News Listening (Expo React Native)

## Publish to Expo Go (Step by Step)

1. Install dependencies

   ```bash
   npm install
   ```

2. Log in to Expo (if needed)

   ```bash
   npx expo login
   ```

3. Start the project and verify it runs

   ```bash
   npx expo start
   ```

4. Publish the update for Expo Go

   ```bash
   npx expo publish
   ```

5. Open Expo Go on your device and scan the QR code, or open the published link from the Expo dashboard.

Notes:
- Use the same Expo account on your device.
- If you update `.env` or config, restart the dev server before publishing.
- For iOS, install Expo Go from the App Store.
